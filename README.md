# AI for Everyone

Marketing site for a beginner-friendly AI course built for practical use in Nigeria, with support for plain English and Nigerian Pidgin.

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion

## Local development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
